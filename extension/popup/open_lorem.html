About

This extension exists to support web developers by providing Lorem Ipsum filler data from various filler data providers.

Development was started by jure965 and then was completely rescripted and reworked by BojanKogoj. We are a fresh IT manager and a web developer, who found out that we need a simple filler content provider as web extension. We looked at various extensions and all were really old plugin style or unsupported by the latest browsers. So we decided to create our own extension in a relatively new and cross platform WebExtension standard.

Since you are reading this, we hope you will find this extension useful. We would love to hear feedback from you. Please rate and comment this extension on addons.mozilla.org.

This extension is open source, its source is available on GitHub.
