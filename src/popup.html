About OpenLorem

This extension exists to support web developers by providing Lorem Ipsum filler text.

We are web developers, who found out that we need a simple filler content provider as web extension. We looked at various extensions and all were really old plugin style or unsupported by the latest browsers. So we decided to create our own extension in a relatively new and cross platform WebExtension standard.

We hope you will find this extension useful. Report bugs, issues or feature requests on GitHub page.

If you like this web extension, consider leaving feedback on addons.mozilla.org.
